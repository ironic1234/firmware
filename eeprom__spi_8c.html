◆ checkVersion()

int checkVersion ( uint16_t version)

: Checks version to ensure app code is new enough

Parameters
dest: Pointer to location to set
len: Length to set
value: Value to set each memory address to
Difference between current and chip versions if app code is newer, -E_V_MISMATCH if app code is old. -E_NO_INIT if memory hasn't been initialized yet

◆ initMem()

int initMem ( GPIO_TypeDef * wc_gpio_port,
uint32_t wc_gpio_pin,
SPI_InitConfig_t * spi,
uint16_t version,
bool force_init )

: Initializes chip metadata. Attemps to read and load all metadata from last run. If memory isn't initialized, force_init can be set to set a default version of 1 and mark the chip as officially in use

Parameters
wc_gpio_port: Mem lock port0
wc_gpio_pin: Mem lock pin
version: Version of app code
force_init: Force initialization if memory isn't initialized. Only set if version is truly 1
